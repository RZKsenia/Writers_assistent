# 📖 Ассистент Писателя

> Одностраничное веб-приложение для организации творческого процесса. Всё необходимое для писателя в одном файле: картотека, планировщик, граф связей и инструменты вдохновения. Работает прямо в браузере, без установки и серверов.

## ✨ Возможности

### 📚 Управление произведениями
- Создание карточек книг/рассказов с жанром, статусом и описанием
- Визуальные статусы: `Черновик` → `В процессе` → `Редактирование` → `Завершено`
- Автоматическая статистика по словам и привязанным персонажам

### 👤 Персонажи и 📍 Места действия
- Детальные карточки с возможностью загрузки фото/иллюстраций внешности
- Поля: описание, внешность, характерные черты, заметки
- Полная переиспользуемость персонажей и локаций в разных произведениях

### 📋 Планировщик глав
- Гибкое управление порядком глав (перемещение вверх/вниз)
- Привязка конкретных персонажей и локаций к каждой главе
- Поле POV (точка зрения) и авторские заметки

### 🔗 Граф связей
- Визуализация взаимоотношений на Canvas
- 10 типов связей: друг, враг, семья, возлюбленные, наставник, соперник, коллега, знакомый, тайная связь, пользовательский тип
- Интерактивный список связей с быстрым редактированием

### 🛠 Инструменты писателя
- 📝 Точный счётчик: слова, символы, без пробелов, предложения, авторские листы (40 000 зн.)
- 🎭 Генератор имён (русские / европейские / фэнтези) с мгновенным копированием
- 💡 Генератор писательских подсказок для преодоления творческого блока
- 🎲 Случайная комбинация «персонаж + место + ситуация» для импровизации

### 💾 Сохранение и перенос
- Автосохранение каждые 30 секунд в `localStorage`
- Ручное сохранение (`Ctrl + S`)
- Экспорт/импорт проекта в JSON для бэкапа или переноса между устройствами
- Глобальный поиск по всем карточкам и главам

## 🚀 Как использовать

1. **Скачайте** файл `index.html` (или клонируйте репозиторий)
2. **Откройте** его в любом современном браузере (Chrome, Firefox, Safari, Edge)
3. Начните создавать произведения, персонажей и планировать сюжет
4. Данные сохраняются автоматически. Для переноса или бэкапа используйте кнопку **«Экспорт JSON»** в нижнем меню

> ⚠️ Приложение работает полностью на стороне клиента. Никакие данные не отправляются на сервер. Для полноценной офлайн-работы скачайте Tailwind и шрифты локально, либо просто используйте при наличии интернета (загружаются через CDN).

## 🛠 Технологии

- Чистый **HTML5 + CSS3 + JavaScript (ES6+)**
- **Tailwind CSS** (через CDN)
- **Canvas API** для отрисовки графа связей
- **Google Fonts**: `Cormorant Garamond`, `Inter`, `JetBrains Mono`
- Zero dependencies, zero build step, один файл
